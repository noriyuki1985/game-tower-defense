# 辺境防衛 v3.12.3

## v3.12.2 / v3.12.3 診断表示・スマホUI最適化

画像生成なし。ステージ選択欄に、各マップの違いを判断しやすくする情報を追加しました。

### 変更点

- `C.stageUiMeta` を追加。
- `meadow / crossroads / ridge` の違いをUI上で表示。
- ステージ選択プルダウンに、ステージ区分・ルート数・難易度を表示。
- ステージ概要カードを追加。
  - 本番候補 / 背景テスト
  - ルート数
  - 難易度
  - 推奨建物
  - ステージ目的
- `stageRuleText` は従来通り地形ルール表示として維持。
- `saveKey` / `audioKey` を `v123` に更新。
- `ASSET_VERSION` を `v3.12.3` に更新。

### ステージ区分

- meadow: 本番候補
- crossroads: 背景テスト
- ridge: 本番候補
