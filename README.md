# 王国防衛 v2.5.3

公開用リポジトリ名: `game-tower-defense`

王だけを操作して、コインを拾い、建設床に投資して城を守るリアルタイム防衛ゲームです。

## 起動方法

`index.html` をブラウザで開きます。

- 既存の王操作、建設、敵進軍、施設攻撃、スマホUIは維持しています。

## 操作

- 移動: WASD / 矢印キー / クリック / タップ
- 一時停止: Space
- 速度変更: F
- 効果音: メニュー内の「効果音」ボタン
- BGM: メニュー内の「BGM」ボタン
- 音量: メニュー内のスライダー

## v2.1.0 の追加内容

新機能は追加せず、今後の開発を続けやすくするためにファイルを分割しました。

- `src/config/game-data.js` にゲーム定義を分離
- `src/config/assets.js` に画像マニフェストと画像読み込み処理を分離
- `src/utils/helpers.js` に汎用関数を分離
- `src/core/save.js` にセーブデータ処理を分離
- `src/game.js` を軽量化
- GitHub Pagesでそのまま動くよう、ビルド不要の通常script構成を維持
- PythonやNodeなどの実行環境は不要

起動方法はこれまで通り `index.html` をブラウザで開くだけです。

## v2.3.0 リファクタリング

ゲーム仕様は変更せず、更新処理を `src/systems/` に分離しました。

- `king-system.js`: 王の移動・被ダメージ
- `wave-system.js`: ウェーブ、ルート、敵出現
- `enemy-system.js`: 敵の移動・特殊行動・撃破処理
- `facility-system.js`: 施設の攻撃・生産・修理
- `soldier-system.js`: 兵士の自動戦闘
- `projectile-system.js`: 弾の移動・着弾
- `build-system.js`: 建設床、投資、強化、発展効果
- `effect-system.js`: 浮遊文字、パーティクル、エフェクト
- `end-system.js`: 勝敗判定、報酬処理

GitHub Pagesでそのまま動くよう、ES Modulesやビルドツールは使っていません。
