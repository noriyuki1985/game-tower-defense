## v3.5.0 5建物強化・新チュートリアル版

- 5施設 x 4レベルの計20枚のレベル別建物画像を追加。

- 分岐強化は一旦使わず、5施設のレベル強化に集中。
- 通常マップは5施設のみで成立する前提を維持。

## v2.2.0 の主要構成

```txt
src/
  game.js
  config/
    game-data.js
    assets.js
  core/
    save.js
  render/
    renderers.js
  utils/
    helpers.js
```

`index.html` は通常の `<script>` 読み込みです。ES Modulesやビルドツールは使っていません。


## v2.3.0 リファクタリング

ゲーム仕様は変更せず、更新処理を `src/systems/` に分離しました。

- `king-system.js`: 王の移動・被ダメージ
- `wave-system.js`: ウェーブ、ルート、敵出現
- `enemy-system.js`: 敵の移動・特殊行動・撃破処理
- `facility-system.js`: 施設の攻撃・生産・修理
- `soldier-system.js`: 兵士の自動戦闘
- `projectile-system.js`: 弾の移動・着弾
- `build-system.js`: 建設床、投資、強化、発展効果
- `effect-system.js`: 浮遊文字、パーティクル、エフェクト
- `end-system.js`: 勝敗判定、報酬処理

GitHub Pagesでそのまま動くよう、ES Modulesやビルドツールは使っていません。

- iPhoneなどのsafe-areaに対応
- PCでは従来通り左パネル表示を維持
- ゲーム仕様、セーブ仕様、GitHub Pages対応は変更なし
